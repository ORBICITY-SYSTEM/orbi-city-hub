# Type Checking - рЃЊрЃарЃЮрЃћрЃЉрЃўрЃЌ рЃњрЃљрЃЏрЃЮрЃарЃЌрЃБрЃџрЃў

## РюЁ рЃарЃљ рЃњрЃљрЃЎрЃћрЃЌрЃЊрЃљ

### 1. Vercel Build Command рЃерЃћрЃўрЃфрЃЋрЃљрЃџрЃљ
- **рЃљрЃЊрЃарЃћ**: `pnpm run build` (Vercel рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃЊрЃљ types-рЃА)
- **рЃљрЃ«рЃџрЃљ**: `pnpm run build:vercel` (type checking-рЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ)

### 2. рЃљрЃ«рЃљрЃџрЃў Scripts рЃЊрЃљрЃћрЃЏрЃљрЃбрЃљ `package.json`-рЃерЃў:
```json
{
  "build": "vite build && esbuild ...",           // Local (рЃарЃЮрЃњрЃЮрЃарЃф рЃљрЃЊрЃарЃћ)
  "build:vercel": "vite build && esbuild ...",     // Vercel (type checking-рЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ)
  "build:check": "tsc --noEmit && vite build ..." // Local type checking-рЃўрЃЌ
}
```

---

## ­Ъћњ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃљ

### РюЁ рЃарЃљрЃбрЃЮрЃЏ рЃљрЃа рЃњрЃљрЃцрЃБрЃГрЃЊрЃћрЃЉрЃљ:

1. **рЃЎрЃЮрЃЊрЃў рЃБрЃЎрЃЋрЃћ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃЊрЃљ**:
   - `esbuild` рЃљрЃа рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃЊрЃљ types-рЃА рЃљрЃЊрЃарЃћ
   - Runtime-рЃќрЃћ рЃЎрЃЮрЃЊрЃў рЃЏрЃБрЃерЃљрЃЮрЃЉрЃЊрЃљ
   - рЃљрЃ«рЃџрЃљ рЃўрЃњрЃўрЃЋрЃћ build process-рЃўрЃљ

2. **Vercel-рЃќрЃћ рЃўрЃњрЃўрЃЋрЃћ build**:
   - `esbuild` transpile-рЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА (TypeScript Рєњ JavaScript)
   - Runtime-рЃќрЃћ JavaScript рЃњрЃљрЃерЃЋрЃћрЃЉрЃБрЃџрЃўрЃљ
   - Type errors рЃљрЃа рЃњрЃљрЃЋрЃџрЃћрЃюрЃљрЃА рЃљрЃ«рЃЊрЃћрЃюрЃА runtime-рЃќрЃћ

3. **Local development**:
   - `dev` script рЃўрЃњрЃўрЃЋрЃћрЃљ
   - `build:check` script-рЃў рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮ type checking-рЃўрЃАрЃЌрЃЋрЃўрЃА

---

## Рџа№ИЈ рЃарЃљ рЃЊрЃљрЃЋрЃЎрЃљрЃарЃњрЃћрЃЌ:

1. **Type Safety**:
   - Type errors рЃљрЃдрЃљрЃа рЃЕрЃљрЃюрЃА build-рЃўрЃА рЃЊрЃарЃЮрЃА
   - IDE-рЃерЃў рЃЎрЃЋрЃџрЃљрЃЋ рЃЕрЃљрЃюрЃА (IntelliSense)

2. **Compile-time Error Detection**:
   - Type mismatches рЃљрЃдрЃљрЃа рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃЉрЃљ build-рЃўрЃА рЃЊрЃарЃЮрЃА
   - Runtime-рЃќрЃћ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў (рЃЏрЃљрЃњрЃарЃљрЃЏ рЃЌрЃБ рЃљрЃЊрЃарЃћ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃЊрЃљ, рЃљрЃа рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ)

---

## ­Ъћё рЃарЃЮрЃњрЃЮрЃа рЃЊрЃљрЃЋрЃљрЃЉрЃарЃБрЃюрЃЮрЃЌ Type Checking:

### рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў 1: Vercel-рЃќрЃћ

### рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў 2: Local-рЃќрЃћ
