## РюЁ рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃљ: Node.js Portable (рЃљрЃЊрЃЏрЃўрЃюрЃўрЃАрЃбрЃарЃљрЃбрЃЮрЃарЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ)


### рЃюрЃљрЃЉрЃўрЃ»рЃў 3: рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћ PATH-рЃерЃў (рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃљрЃЏ рЃАрЃћрЃАрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА)


## ­Ъџђ Alternative: fnm (Fast Node Manager) - рЃЏрЃўрЃарЃЕрЃћрЃЋрЃюрЃўрЃљ!

`fnm` рЃљрЃарЃўрЃА Node.js version manager, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф **рЃљрЃа рЃАрЃљрЃГрЃўрЃарЃЮрЃћрЃЉрЃА admin рЃБрЃцрЃџрЃћрЃЉрЃћрЃЉрЃА**!

### рЃюрЃљрЃЉрЃўрЃ»рЃў 1: рЃЕрЃљрЃЏрЃЮрЃбрЃЋрЃўрЃарЃЌрЃћ fnm

1. рЃњрЃљрЃЊрЃљрЃЊрЃў: **https://github.com/Schniz/fnm/releases/latest**
2. рЃўрЃърЃЮрЃЋрЃћ: **`fnm-windows.zip`** (Windows x64)
3. рЃЕрЃљрЃЏрЃЮрЃбрЃЋрЃўрЃарЃЌрЃћ рЃЊрЃљ рЃњрЃљрЃ«рЃАрЃћрЃюрЃў ZIP рЃцрЃљрЃўрЃџрЃў

### рЃюрЃљрЃЉрЃўрЃ»рЃў 2: рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћ fnm (рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃЊрЃЮрЃюрЃћрЃќрЃћ)

рЃњрЃљрЃ«рЃАрЃћрЃюрЃў PowerShell рЃЊрЃљ рЃерЃћрЃљрЃАрЃарЃБрЃџрЃћ:

```powershell
# рЃњрЃљрЃЊрЃљрЃЊрЃў рЃЕрЃљрЃЏрЃЮрЃбрЃЋрЃўрЃарЃЌрЃБрЃџ fnm рЃцрЃЮрЃџрЃЊрЃћрЃарЃерЃў
cd Downloads

# рЃњрЃљрЃ«рЃАрЃћрЃюрЃў fnm-windows.zip (рЃЌрЃБ рЃБрЃЎрЃЋрЃћ рЃњрЃљрЃ«рЃАрЃюрЃўрЃџрЃў рЃњрЃљрЃЦрЃЋрЃА, рЃњрЃљрЃЊрЃљрЃЊрЃў рЃерЃўрЃњрЃюрЃўрЃЌ)
cd fnm-windows

# рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћ fnm PATH-рЃерЃў (рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃљрЃЏ рЃАрЃћрЃАрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА)
$env:PATH = "$PWD;$env:PATH"

# рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћ fnm (рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃЊрЃЮрЃюрЃћрЃќрЃћ)
.\fnm.exe install --skip-shell

# рЃњрЃљрЃљрЃЦрЃбрЃўрЃБрЃарЃћ fnm (PowerShell-рЃўрЃАрЃЌрЃЋрЃўрЃА)
fnm env --use-on-cd | Out-String | Invoke-Expression

# рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћ Node.js LTS

# рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћ LTS рЃЋрЃћрЃарЃАрЃўрЃљ
fnm use --lts

# рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћ
node --version
npm --version
```

### рЃюрЃљрЃЉрЃўрЃ»рЃў 3: рЃњрЃљрЃљрЃЎрЃћрЃЌрЃћ fnm рЃЏрЃБрЃЊрЃЏрЃўрЃЋрЃў (PowerShell Profile-рЃерЃў)

```powershell

# рЃњрЃљрЃЊрЃљрЃбрЃЋрЃўрЃарЃЌрЃћ PowerShell рЃљрЃю рЃерЃћрЃљрЃАрЃарЃБрЃџрЃћ:
```
