თუ **Google Domains** გაქვთ:
```
1. domains.google.com → orbicitybatumi.com
2. DNS → Manage Custom Records
3. დაამატეთ CNAME record:
   - Host: team
   - Type: CNAME
   - Data: orbi-city-hub-production.up.railway.app
   - TTL: 1h
4. Save
```

თუ **Cloudflare** გაქვთ:
```
1. cloudflare.com → orbicitybatumi.com
2. DNS → Add Record
3. Type: CNAME
4. Name: team
5. Target: orbi-city-hub-production.up.railway.app
6. Proxy Status: Proxied (🟠 Orange Cloud)
7. Save
```

## Troubleshooting

### Problem 1: Build Failed
**Error:** `Build failed with exit code 1`

**Solution:**
```bash
# Check package.json scripts
"scripts": {
  "build": "tsc && vite build",
  "start": "node server/_core/index.js"
}

# Verify Node version
"engines": {
  "node": ">=18.0.0"
}
```

### Problem 2: Database Connection Failed
**Error:** `ECONNREFUSED` or `Access denied`

**Solution:**
```bash
# Check DATABASE_URL format
DATABASE_URL=mysql://user:password@host:port/database

# Verify Railway MySQL is running
Railway Dashboard → MySQL → Status: Running ✅
```

### Problem 3: Environment Variables Not Working
**Error:** `process.env.GEMINI_API_KEY is undefined`

**Solution:**
```bash
# Railway Dashboard → Variables
# Make sure all variables are set
# Redeploy after adding variables:
Railway Dashboard → Deployments → Redeploy
```

### Problem 4: Custom Domain Not Working
**Error:** `DNS_PROBE_FINISHED_NXDOMAIN`

**Solution:**
```bash
# Check DNS propagation
https://dnschecker.org

# Verify CNAME record
dig team.orbicitybatumi.com CNAME

# Wait 5-60 minutes for DNS propagation
```

### Problem 5: Port Binding Error
**Error:** `Error: listen EADDRINUSE: address already in use :::3000`

**Solution:**
```javascript
// server/_core/index.ts
const PORT = process.env.PORT || 3000;
app.listen(PORT, '0.0.0.0', () => {
  console.log(`Server running on port ${PORT}`);
});
```

Railway ავტომატურად აყენებს `PORT` environment variable-ს.
